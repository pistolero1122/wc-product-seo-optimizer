🛠️ WooCommerce Product SEO Optimizer (MVP)
The Problem
Many e-commerce merchants struggle with inconsistent product data, which leads to poor search engine rankings and low conversion rates.

The Solution
This lightweight plugin provides an instant SEO Quality Score directly on the product edit page. It acts as a real-time auditor, giving merchants immediate feedback on what needs to be fixed.

Key Metrics Analyzed
Title Optimization: Checks for optimal length (50–80 chars).

Content Depth: Analyzes description length for better indexing.

Media Audit: Ensures a minimum of 3 high-quality images.

Metadata Check: Scans for existing Meta Descriptions from Yoast/AIOSEO.

Price Validation: Ensures products aren't published without a price.

Installation
Download the wc-optimizer.php file from this repository.

Upload it to your WordPress site under Plugins -> Add New.

Activate the plugin and go to any Product page to see your score.
